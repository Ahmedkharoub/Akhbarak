# Akhbarak Application

## Overview

This application displays news articles from our backend, providing users with a convenient way to stay updated on the latest news across different categories and topics.

## Features

- **News Highlights**: Quickly check latest news in a nice carousel.
- **Customizable Feeds**: Personalize your news feed by selecting preferred language.
- **Catchy Visuals**: Entertain your eyes with article images while reading.

## Technologies Used

- **Mobile Platform**: [Android](https://www.android.com)
- **Image Loader**: [Picasso](https://square.github.io/picasso/)
- **Network**: [Retrofit](https://square.github.io/retrofit/)
- **Version Control**: [Git](https://git-scm.com/)
- **Backend**: Spring Boot ([@mohkamfer](https://github.com/mohkamfer/))
- **Deployment**: ([Hetzner](https://www.hetzner.com/) ([@mohkamfer](https://github.com/mohkamfer/)))

## Usage

1. **Browse News**: Navigate through different news categories.
2. **Read Article**: Tap on a category card to view and read the article.
3. **Change Language**: Tap on the overflow button, go to settings, and choose your desired language.
